# 저작권 안내

## RSS 피드 사용에 대한 안내

본 사이트는 AI타임스의 공개 RSS 피드를 사용하여 기사 정보를 수집합니다.

### 사용 방식
- **제목, 링크, 요약만 표시**: RSS 피드에 포함된 제목과 요약만 사용합니다.
- **본문은 원본 링크로 이동**: 전체 본문은 원본 사이트로 이동하여 확인할 수 있습니다.
- **출처 명시**: 모든 기사에 출처(AI타임스)를 명확히 표시합니다.

### 저작권
- 모든 기사의 저작권은 원본 출처(AI타임스)에 있습니다.
- 본 사이트는 기사 정보를 링크 형태로 제공하며, 원본 사이트로의 트래픽을 유도합니다.
- RSS 피드는 공개적으로 제공되는 표준 형식이며, 뉴스 리더나 검색 엔진에서 일반적으로 사용하는 방식입니다.

### robots.txt 확인
- AI타임스 robots.txt: User-agent: * 허용, /admin/만 차단
- 본 크롤러는 robots.txt를 준수합니다.

### 문의
저작권 관련 문의사항이 있으시면 원본 사이트(AI타임스)에 직접 문의하시기 바랍니다.
